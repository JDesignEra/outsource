Initial Setup
------
#### Install Required Packages
```cmd
npm i
```


How to Run?
------
```cmd
node app.js
```

Commit Legends
------
| Format | Description |
|--------|-------------|
| + | Added |
| * | Updated |
| - | Removed |
| { Variable / Code } | Variable / Code related
| \[ File ] | File (e.g. style.css) |
| ( Folder ) | Folder / Directory |
| < Notes > | Notes / Things to note |
